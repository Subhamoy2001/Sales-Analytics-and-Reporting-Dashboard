# Sales Analytics and Reporting Dashboard

## Overview
This project is an interactive Sales Analytics and Reporting Dashboard developed using Microsoft Excel and Power BI. The dashboard helps analyze sales performance, identify trends, and generate business insights for better decision-making.

---

## Features
- Sales performance analysis
- Revenue and profit tracking
- Category-wise and region-wise analysis
- Interactive charts and visualizations
- Data cleaning and preprocessing
- Business reporting dashboard

---

## Tools & Technologies
- Microsoft Excel
- Power BI

---

## Key Insights
- Identified top-performing products and categories
- Analyzed monthly sales trends
- Compared regional sales performance
- Generated reports for business analysis

---

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Analysis
4. Dashboard Creation
5. Business Insight Generation
